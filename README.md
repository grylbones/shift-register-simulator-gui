# Shift Register Simulator (Python OOP + Tkinter)

A visual, interactive simulator for 4-bit shift registers built using Object-Oriented Programming and Tkinter. 

This project implements and visualizes:

* SISO – Serial-In Serial-Out

* SIPO – Serial-In Parallel-Out

* PISO – Parallel-In Serial-Out

* PIPO – Parallel-In Parallel-Out

Each register type is built using a clean OOP structure with an abstract base class and multiple polymorphic child classes.

## Features

* Fully interactive Tkinter GUI

* Real-time D/Q updates

* Clock pulse simulation

* Serial & parallel input modes

* Visual hardware-style diagram

* Clean OOP architecture (inheritance + polymorphism)

* Main menu for selecting register type

## Tech Stack
* Python

* Tkinter

* OOP (Inheritance, Abstraction, Polymorphism)
